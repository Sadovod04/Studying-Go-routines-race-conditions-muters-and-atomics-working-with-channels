# Studying Go routines, race conditions, muters and atomics, working with channels

Учебное приложение на Go, демонстрирующее работу с параллельностью и конкурентностью.
Проект показывает, как использовать горутины, каналы, мьютексы и атомарные операции для безопасного выполнения кода в многопоточной среде.

В примерах рассматриваются ситуации с гонками данных, способы их обнаружения и устранения с помощью инструментов синхронизации и коммуникации между горутинами.

## Возможности

* Работа с горутинами (goroutines)
* Демонстрация и устранение гонок данных (data races)
* Использование мьютексов (sync.Mutex, sync.RWMutex)
* Применение атомарных операций (sync/atomic)
* Обмен данными через каналы (channels)
* Безопасная синхронизация доступа к общим ресурсам

## Требования

Go 1.18+
